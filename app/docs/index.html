Introducing BVBootstrap.

Why use Bootstrap at Bazaarvoice?

Loosely-coupled teams.

Awesome for innovation, not so awesome for the user experience. Bootstrap is both a living example and the glue to keep our R&D department on-brand and on-target so client users get a consistent experience.

Responsive and retina-ready

With responsive options for page layouts, responsive helper classes, and font-driven icons out of the box, this is an easiest way to get your app minimally responsive. It shouldn't replace continuing education and thoughtful design, but it's a start.

Thanks, @mdo and @fat

Twitter's project is incredibly well-documented and developed. Read it if you haven't already.

Plus, teams around the company (and the world) are already using it. We'll do our best to make sure Bootstrap markup is compatible with BV Bootstrap with no effort on your part, so you can work in a familiar framework or even swap out the resources on an existing project and instantly be in sync with the rest of the BV client-facing ecosystem.
